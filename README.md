## 🛠️ CORE SYSTEM ARCHITECTURE
Portal ini dirancang untuk menigkatkan literasui politik remaja dan kesadaran siswa terhadap berita hoax. Dibangun dengan fokus pada aksesibilitas mobile menggunakan modalitas AI dan efisiensi satu file.

### 🚀 Key Features
* **Galactic Interface:** UI bertema Cyber-Neon dengan animasi partikel bintang dan glitch modern.
* **NASA APOD Integration:** Menampilkan data citra satelit terbaru langsung dari API NASA secara real-time.
* **Emergency Protocol (404):** Sistem proteksi member dengan fitur *Auto-Warp Redirect* dan animasi Black Hole.
* **Floating Navigation:** Tombol akses cepat (FAB) yang ergonomis untuk navigasi admin dan pelaporan bug.

- HTML5
- CSS3 (Flexbox & Grid)
